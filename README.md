# AdminGenerator
skeleton admin structure generator.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "admin_generator", github: "departure-inc/admin-generator"
```

## Usage
How to use my plugin.

```bash
$ bundle exec rails g admin Thing
```
